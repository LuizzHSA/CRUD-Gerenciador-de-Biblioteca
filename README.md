📚 CRUD Biblioteca — Electron + SQLite + Bootstrap

Este é um projeto de **CRUD de Livros** com interface desktop, desenvolvido com **Electron**, **SQLite** como banco de dados local e **Bootstrap** para o estilo visual.


🛠️ Tecnologias Utilizadas

- [Electron](https://www.electronjs.org/) — Para criar apps desktop com tecnologias web
- [SQLite3](https://www.sqlite.org/) — Banco de dados local, leve e sem servidor
- [Bootstrap 5](https://getbootstrap.com/) — Estilização responsiva e moderna
- JavaScript (Frontend)
- HTML5 e CSS3



📦 Funcionalidades

- ✅ Cadastrar livros
- 📋 Listar livros cadastrados
- 🗑️ Deletar livros
- 💾 Salvar os dados localmente com SQLite
- 🧪 App 100% offline (não precisa de internet ou servidor)
- ⚡ Interface leve, responsiva e simples



💻 Como executar o projeto

1. Clone o repositório

git clone https://github.com/seu-usuario/crud-biblioteca.git
cd crud-biblioteca


2. Instale as dependências

npm install

3. Rode o projeto

npm start
